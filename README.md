# 📌 산업별 디지털·AI 활용 필요성 분석

---

## 🟢 1. 개요

| 항목 | 내용 |
| --- | --- |
| **데이터 파일명** | `인공지능기술_AI필요성.xlsx` |
| **분석 방식** | 산업별 AI 필요성 비율 계산 및 그래프 시각화 |
| **사용 프로그램** | Python GUI (Tkinter) |

---

## 🟦 2. 연구 주제 및 목적

디지털 전환과 AI 기술 도입의 확산 속에서 산업별 종사자가 인공지능의 필요성을 어떻게 인식하고 있는지 분석한다.

본 연구의 목적은 다음과 같다:

- 산업별 **AI 필요성 인식 수준 비교**
- 디지털 전환 전략 수립을 위한 기초 자료 제공
- AI 도입 인식이 낮은 산업의 **정책·교육적 지원 필요성 확인**

---

## 🟧 3. 시스템 구조

```
[ Excel 데이터 ]
        │
        ▼
데이터 로딩 및 전처리 (pandas)
 - 산업 구분 / 필요할 비율 / 필요하지 않음 비율
 - NaN 처리 및 숫자형 변환
        │
        ▼
AI 필요성 분석 알고리즘
 - 필요함 = 매우 필요 + 약간 필요
 - 필요하지 않음 = 별로 + 전혀
        │
        ▼
Tkinter GUI
 - 산업 선택 드롭다운
 - 분석 결과 텍스트 출력
 - 그래프 시각화

```

## 🟨 4. 사용 라이브러리 및 API 정보

| 라이브러리 | 용도 |

## 🔍 6. 결론 및 시사점

### ✔ 결론

산업별로 AI 필요성 인식 수준의 차이가 뚜렷하며 디지털 전환에 대한 준비 정도가 다름을 확인할 수 있다.

### ✔ 시사점

| 시사점 | 설명 |
| --- | --- |
| AI 필요성이 높은 산업 | 지속적인 기술 투자로 경쟁력 강화 필요 |
| 필요성 인식이 낮은 산업 | 인식 확산을 위한 정책·교육 지원 필요 |
| 후속 연구 방향 | 기업 규모/지역/직무별 AI 인식 분석 확장 가능 |
